1. RUST BACKEND LIBRARIES (The "Engine Room")
--------------------------------------------------------------------------------
A. Web & Core Asynchronous Logic:
- actix-web (v4.0): A world-class, extremely fast, type-safe web framework.
- tokio (v1.0): The asynchronous runtime that executes your Rust tasks.
- tokio-tungstenite: High-performance WebSocket implementation for game sync.

B. Authoritative Chess Logic:
- shakmaty (v0.26): A state-of-the-art chess library that uses bitboards for 
  ultra-fast move validation and FEN parsing.
- chess (v3.2): High-performance move generation used for AI calculations.
- board-game-traits: Standardizes game state handling for the engine.

C. Persistence Layer:
- sqlx (v0.7): Type-safe, asynchronous SQL toolkit for PostgreSQL. 
- redis (v0.24): In-memory data structure store for caching active game sessions.

D. Monitoring & Performance Profiling:
- sentry (v0.32): SDK for automatic crash reporting and error tracking.
- dhat (v0.3): Memory profiler used to detect heap allocation issues.
- tokio-console: Real-time diagnostics for async task scheduling.
- prometheus: Metrics collection for active player counts and move latency.

E. Security:
- jsonwebtoken (v9.3): Securely signs and verifies user game tokens.
- bcrypt (v0.15): Industry-standard password hashing (backend-side).
